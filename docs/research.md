## 2. Почему мало продаж — диагностика

### 2.1 Системные причины (применимо к нашей ситуации)

По данным исследований 2025 года, низкие продажи курсов почти никогда не связаны с качеством контента. Реальные причины:

1. **Нет аудитории перед запуском.** Успешные авторы курсов имеют прогретую аудиторию ДО старта продаж. Бот без трафика = пустая касса.

2. **Продаётся информация, а не трансформация.** В 2025 году "6 уроков + пожизненный доступ" не работает. Покупают результат: "Через 3 дня ты сделаешь свой первый AI-ролик".

3. **Слабый онбординг в боте — drop-off на 3–4 шаге.** Исследования показывают: большинство отказов происходит после 3-го сообщения, если не создан интерес. Один кейс: пересмотр последовательности сообщений поднял продажи на 22%.

4. **Нет социального доказательства.** 98% покупателей читают отзывы перед покупкой. Бот без отзывов и примеров работ = холодный старт.

5. **Нет urgency/scarcity.** Без дедлайна пользователь откладывает покупку навсегда.

6. **Нет лид-магнита.** Прямая продажа холодному пользователю — низкий конверт. Нужен бесплатный "крючок".

7. **Неправильное ценообразование.** Слишком дешёво = нет доверия. Слишком дорого без обоснования = нет покупки.

8. **Плохой трафик или его полное отсутствие.** Бот сам по себе не привлекает новых пользователей.

### 2.2 Специфика нашего бота

Текущий флоу: `/start` → кнопки "О курсе" / "Купить" → принятие оферты → ввод email → ссылка на YooKassa → оплата → доступ.

**Проблемы флоу:**
- Сразу предлагается купить — нет прогрева, нет доверия.
- Нет примеров работ, нет результатов студентов.
- Нет ограничения по времени/количеству мест.
- Нет бесплатного фрагмента контента.
- Email собирается слишком рано в воронке (раздражает).
- Нет повторного касания: если не купил сейчас — навсегда потерян.

## 3. Конверсионные бенчмарки (Telegram)

| Этап воронки | Норма |
|---|---|
| Реклама → подписка/старт бота | 5–15% |
| Активация бота после подписки | 50–70% |
| Прохождение прогрева (3–5 сообщений) | 30–50% |
| Конверсия в покупку (из прогрева) | 3–12% |
| Открытие сообщений в боте vs email | 4–5x выше |
| Конверсия воронки в мессенджере vs email | 3–4x выше |

Конверсия в подписку через бота: **90–95%** (против 30–50% в канал). Это ключевое преимущество формата.

## 4. Best practices: что реально работает

### 4.1 Структура воронки продаж курса в Telegram

```
[Трафик: реклама в каналах / Telegram Ads / сарафан]
         ↓
[Лид-магнит: бесплатный урок / чек-лист / мини-демо]
         ↓
[Прогревающая последовательность (3–5 сообщений):
  - Боль ЦА
  - Результат студентов / соцдоказательства
  - Демонстрация продукта
  - Ограниченное предложение / urgency]
         ↓
[Кнопка "Купить" → оплата → мгновенный доступ]
         ↓
[Post-purchase: напоминания, допродажа, сбор отзывов]
```

### 4.2 Ценообразование

- **Психологически выгодные точки:** 990₽, 1 990₽, 4 990₽, 9 900₽ (не круглые числа).
- **Якорная цена:** показать "старую цену" перед скидкой — работает при наличии реального обоснования.
- **Tiered pricing (65% успешных курсов):** Базовый / Стандарт / Премиум.
  - Базовый: только уроки
  - Стандарт: уроки + материалы + чек-листы
  - Премиум: уроки + материалы + разбор работы + доступ в закрытый чат
- **Completion rate выше при цене >200$**: пользователи воспринимают дешёвые курсы как малоценные и не доходят до конца.

### 4.3 Лендинг / страница курса (если добавить веб-лендинг)

1. **Видео-превью** выше fold (60–90 сек, без автозапуска со звуком) — поднимает конверсию до +80%.
2. **Боль → Трансформация** в заголовке (не "6 уроков", а "Через 3 дня ты создашь свой первый AI-ролик").
3. **Социальные доказательства** — скриншоты работ студентов, видео-отзывы, цифры.
4. **Таймер urgency** — ограничение по времени или количеству мест.
5. **Гарантия возврата** — снижает барьер входа.
6. **FAQ блок** — закрывает типичные возражения.
7. **Чёткий единственный CTA**: одна яркая кнопка "Начать обучение".

### 4.4 UX онбординга в боте (что снижает drop-off)

- Короткие сообщения (не "простыни" текста).
- Кнопки вместо текстового ввода везде, где можно.
- Встроенные preview работ (GIF, короткое видео) сразу в боте.
- Персонализация: "Привет, [имя]!" работает лучше безликого обращения.
- Геймификация: прогресс, шаги, бейджи.
- **Email собирать ПОСЛЕ покупки** или при запросе чека, не в начале воронки.
- Повторные касания: если пользователь не купил — напомнить через 24ч, через 72ч с другим оффером.

### 4.5 Социальные доказательства (критично)

- **98% покупателей** изучают отзывы перед покупкой.
- Лучший формат: видео-отзыв с конкретным результатом ("До: не умел / После: вот мой ролик").
- В боте: встраивать примеры работ студентов прямо в прогревающие сообщения.
- Счётчик покупателей ("Уже 143 человека прошли курс") — работает.

## 5. Telegram Mini App vs Bot — стратегия

| Критерий | Bot (текущее) | Mini App |
|---|---|---|
| UX | Чат, кнопки | Полноценный UI, как мобильное приложение |
| Разработка | Сделано ✓ | Требует веб-разработки |
| Конверсия | +23% (AI-боты) | Выше за счёт deep linking и плавного UX |
| Платежи | YooKassa (внешний) | Telegram Stars (внутренний, без трений) |
| Лучше для | Воронка, автоматизация | Каталог, кабинет студента, просмотр |

**Рекомендация:** Bot как воронка + Mini App как личный кабинет студента — комбо даёт максимум. В краткосрочной перспективе: улучшить бот. В среднесрочной: добавить Mini App для просмотра курса.

## 6. Трафик — откуда брать пользователей

Без трафика бот не продаёт. Основные каналы для русскоязычной аудитории:

1. **Telegram-каналы (посев)** — покупка рекламных постов в тематических каналах (AI, нейросети, контент, дизайн, SMM, заработок). Лучший ROI для старта.
2. **Telegram Ads** (официальная реклама) — работает, но минимальный бюджет €2 CPM.
3. **ВКонтакте / Shorts / Reels** — короткие демо-видео с результатами (сам контент = реклама).
4. **YouTube Shorts** — демонстрация "До/После" AI-анимации набирает органику.
5. **Реферальная программа** — текущие студенты приводят друзей за бонус/скидку.
6. **Коллаборации** — обмен постами с другими авторами схожей тематики.

**Ключевой инсайт:** Контент-маркетинг через короткие AI-видео ("смотри, что я сделал за 5 минут") — лучший способ бесплатно привлекать горячую аудиторию в 2025–2026.

## 8. Ключевые инсайты — итог

### Почему мало продаж (вероятные причины, по убыванию вероятности)

1. **Нет трафика** — бот существует, но никто не знает о курсе.
2. **Нет прогрева** — пользователь приходит холодным и сразу видит кнопку "Купить".
3. **Нет социальных доказательств** — нет отзывов, нет примеров работ студентов.
4. **Нет urgency** — можно купить в любое время, значит куплю потом (никогда).
5. **Нет лид-магнита** — нет причины дать боту свои данные бесплатно.
6. **Нет повторных касаний** — не купил сразу = навсегда потерян.

### Что нужно сделать (по приоритету)

#### Быстрые wins (в боте, без лендинга):
- [ ] Добавить прогревающую последовательность (3–5 сообщений перед кнопкой "Купить").
- [ ] Встроить примеры работ студентов (видео/GIF прямо в боте).
- [ ] Добавить urgency ("Цена поднимается XX числа" / "Осталось N мест").
- [ ] Добавить бесплатный лид-магнит (1 урок бесплатно / чек-лист / мини-демо).
- [ ] Переместить сбор email на шаг после намерения купить.
- [ ] Добавить delayed reminders (24ч и 72ч после первого контакта).

#### Средний приоритет:
- [ ] Создать полноценный лендинг (веб-страница с видео, отзывами, CTA).
- [ ] Настроить SEO для лендинга.
- [ ] Добавить tiered pricing (2–3 тарифа).
- [ ] Настроить трафик-канал (хотя бы один: посев в Telegram-каналах).

#### Долгосрочно:
- [ ] Telegram Mini App как кабинет студента.
- [ ] Контент-маркетинг (короткие видео с результатами).
- [ ] Реферальная программа.
- [ ] Добавить новые инструменты в курс (Kling 2.6 — сейчас #1 по соотношению цена/качество).

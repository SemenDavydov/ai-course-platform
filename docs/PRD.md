> Дата: март 2026
> Основа: [docs/research.md](./research.md)

## Контекст продукта

**Курс:** «Экспресс курс по созданию ИИ анимаций и изображений»
**Инструменты в курсе:** Grok (генерация AI-видео) + CapCut (монтаж)
**Цена:** 3 990 ₽
**Трафик:** Instagram (основной), TikTok (вторичный)
**Текущая проблема:** мало продаж, несмотря на работающий бот и платформу

---

## Цель

Перейти от единичных покупок к стабильным продажам.
**Метрика успеха:** конверсия бот → покупка достигает 3–5% (сейчас предположительно <1%).

---

## Приоритизация

| # | Фича | Приоритет | Трудозатраты | Ожидаемый эффект |
|---|------|-----------|--------------|------------------|
| 1 | Прогревающая последовательность в боте | 🔴 Высокий | S (1 день) | +15–25% конверсии |
| 2 | Лид-магнит (чек-лист PDF) | 🔴 Высокий | S (0.5 дня) | +20–30% лидов |
| 3 | Email после намерения купить | 🔴 Высокий | S (0.5 дня) | -30% drop-off |
| 4 | Delayed reminders 24ч / 72ч | 🔴 Высокий | M (1–2 дня) | +10–20% конверсии |
| 5 | Лендинг (HTML-страница в FastAPI) | 🟡 Средний | L (3–5 дней) | Новый канал трафика |
| 6 | SEO для лендинга | 🟡 Средний | S (1 день) | Органика через 2–4 мес |
| 7 | Трафик из Instagram и TikTok | 🟡 Средний | XL (ongoing) | Основа роста |

> **Тарифы (tiered pricing):** отложено. Сначала нужно отладить воронку и стабилизировать продажи. Вернуться к этому вопросу, когда будет 20+ продаж в месяц.

## Feature 1 — Прогревающая последовательность в боте

### Проблема
Текущий флоу: `/start` → «О курсе» / «Купить» — нет прогрева, пользователь приходит холодным.
Drop-off при первом контакте без прогрева: >70%.

### Решение
Заменить немедленное предложение купить на 4-шаговую последовательность перед CTA.

### Новый флоу бота

```
/start
  → [Сообщение 1] Приветствие + боль ЦА
    Текст: «Привет, {имя}! 👋
    Хочешь научиться делать крутые AI-видеоролики и анимации — с нуля, без сложных программ?
    Я покажу, как это делается в Grok и CapCut — буквально за несколько часов практики.»
    [Кнопка] "Да, хочу уметь так же →"

  → [Сообщение 2] Что конкретно научишься
    Текст: «В курсе 6 уроков. После прохождения ты умеешь:
    ✅ Генерировать AI-видео и анимации в Grok
    ✅ Монтировать и доводить до результата в CapCut
    ✅ Делать ролики, которые выглядят профессионально

    Всё это без дизайнерского образования и дорогих программ.»
    [Кнопка] "Что внутри курса? →"

  → [Сообщение 3] Программа + демо
    Текст: список уроков с кратким описанием + короткое демо-видео (что умеет делать Grok)
    [Кнопки] "Получить чек-лист бесплатно 🎁" | "Купить курс — 3 990 ₽ →"

  → [Кнопка "Купить курс"] → существующий флоу оплаты (с переработанным порядком — см. F3)
```

### Urgency
Ограничение мест — нелогично на текущем этапе (мало трафика). Urgency пока не используем.
Вернуться к этому вопросу после стабилизации трафика (50+ новых пользователей/нед).

### Технические требования
- Файл: `app/bot/bot.py`
- Новые FSM-состояния не нужны — линейная последовательность через `InlineKeyboardMarkup`
- Кнопка «Купить курс» ведёт в существующий флоу оплаты (после принятия оферты и email)

### Контент (нужно подготовить автору)
- [ ] Короткое демо-видео (15–30 сек) — что можно сделать в Grok (отправить как `video` в Telegram)
- [ ] Текущее число «мест в потоке» для urgency

## Feature 2 — Лид-магнит: чек-лист

### Проблема
Прямая продажа холодному пользователю — низкий конверт. Нужен бесплатный «крючок».

### Решение
Бесплатный PDF-чек-лист как точка входа в воронку перед покупкой.

### Содержание чек-листа
**«Топ-10 типов AI-видео, которые сейчас набирают миллионы просмотров»**
- Формат: 1–2 страницы, PDF, с брендингом курса
- Для каждого типа: короткое описание + зачем это работает
- В конце: «Как научиться делать такие видео самому → [ссылка на бот]»

### Флоу
```
Кнопка "Получить чек-лист бесплатно 🎁"
  → Бот отправляет PDF (send_document)
  → Через 90 секунд — follow-up:
    «Кстати, в курсе я показываю, как создавать именно такие форматы —
     с нуля, в Grok и CapCut. Хочешь посмотреть программу?»
  → [Кнопка] "Посмотреть программу →" → Сообщение 3 прогрева (программа + CTA)
```

### Технические требования
- Файл чек-листа: `app/static/checklist.pdf`
- Отправка: `await bot.send_document(chat_id, FSInputFile("app/static/checklist.pdf"))`
- Follow-up: Celery-таск `send_lead_magnet_followup(telegram_id, delay=90)` (надёжнее `asyncio.sleep`)

### Контент (нужно подготовить)
- [ ] PDF чек-лист (1–2 страницы) с брендингом и ссылкой на бот

## Feature 3 — Email после намерения купить

### Проблема
Сейчас email запрашивается до показа ссылки на оплату. Пользователь ещё не принял решение — раздражает, увеличивает drop-off.

### Решение
Перенести запрос email на шаг ПОСЛЕ нажатия «Купить» — непосредственно перед созданием платежа.

### Новый порядок
```
[Кнопка "Купить курс"]
  → Принятие оферты (оставить как есть)
  → Сообщение: «Для выставления чека по закону РФ нужен email.
                 Укажи его — и получишь ссылку на оплату:»
  → FSM: Form.waiting_for_email (существующий, без изменений)
  → Создание платежа → ссылка YooKassa
```

### Технические требования
- Файл: `app/bot/bot.py`
- Минимальные изменения: переставить вызов `waiting_for_email` после принятия оферты
- Добавить поясняющий текст перед запросом email (снимает возражение)

## Feature 4 — Delayed Reminders (24ч / 72ч)

### Проблема
Если пользователь не купил сразу — он навсегда потерян. Повторного касания нет.

### Решение
Автоматические follow-up через 24 и 72 часа после первого контакта без покупки.

### Сообщение 24ч (без давления, с пользой)
```
Привет, {имя}! 👋

Вчера ты заходил в наш курс по AI-видео.

Решил поделиться одной фишкой: в Grok есть функция, которую большинство
не использует — но она даёт совершенно другой уровень анимации.
Разбираю её в уроке 3.

Курс всё ещё доступен за 3 990 ₽ →

[Кнопка] "Посмотреть курс →"
```

### Сообщение 72ч (с urgency через ограниченность мест)
```
{Имя}, последний раз пишу 😊

В потоке осталось несколько мест. После заполнения — закрываю запись
до следующего набора.

Если хочешь создавать AI-ролики — сейчас хорошее время войти.

3 990 ₽ → [Кнопка] "Купить курс →"   [Кнопка] "Не интересует"
```

### Технические требования

**Миграция БД** — добавить в модель `User`:
```python
first_contact_at: Mapped[Optional[datetime]] = mapped_column(DateTime, nullable=True)
reminder_24h_sent: Mapped[bool] = mapped_column(Boolean, default=False)
reminder_72h_sent: Mapped[bool] = mapped_column(Boolean, default=False)
```

**Alembic:** новая миграция `add_reminder_fields_to_user`

**Bot (`app/bot/bot.py`):** при `/start` записывать `user.first_contact_at = datetime.utcnow()` если поле `None`

**Celery task (`app/tasks.py`):**
```python
@celery_app.task
def send_reminders():
    # Найти пользователей: first_contact_at + 24h < now, has_access=False, reminder_24h_sent=False
    # Отправить сообщение, установить reminder_24h_sent=True
    # Аналогично для 72h
```

**Beat schedule (`app/celery_app.py`):** добавить `send_reminders` каждый час

**Opt-out:** кнопка «Не интересует» → `reminder_72h_sent = True` (больше не беспокоить)

## Feature 5 — Лендинг (веб-страница)

### Проблема
Нет SEO-индексируемой точки входа. Весь трафик идёт только через бот.
Нельзя давать ссылку «на сайт» в Instagram bio или TikTok описании.

### Решение
Полноценный HTML-лендинг на главной странице `/`.

### Структура (секции сверху вниз)

#### 1. Hero
- **H1:** «Создавай AI-видеоролики и анимации с нуля — за несколько часов»
- **Подзаголовок:** «Практический экспресс-курс: Grok + CapCut. Без опыта, без сложных программ.»
- **Демо-видео:** встроенное видео с примером того, что умеет делать Grok (15–30 сек, Kinescope embed или прямой файл)
- **CTA:** «Начать обучение — 3 990 ₽ →» (ведёт на бот)
- Urgency под кнопкой: не добавлять пока нет реального дедлайна или обоснования

#### 2. Блок «Боль → Решение»
- «Видишь крутые AI-ролики и не знаешь, как это делается?»
- «Пробовал разобраться — но инструментов слишком много, непонятно с чего начать?»
- «Кажется, что это для профессионалов, а не для тебя?»
- → «Этот курс — для тех, кто хочет научиться раз и навсегда»

#### 3. Блок «Что ты умеешь после курса»
Конкретные навыки-результаты (не «6 уроков»):
- Генерировать AI-видео и анимации в Grok за 10–15 минут
- Монтировать и доводить ролик до финала в CapCut
- Создавать контент, который выглядит профессионально
- Использовать AI как инструмент — быстро и эффективно

#### 4. Программа курса
Список уроков с кратким описанием каждого

#### 5. О преподавателе
Короткий текст + фото (личный бренд важен для авторских курсов)

#### 6. Лид-магнит на лендинге
- «Получи чек-лист бесплатно» → кнопка ведёт в бот с параметром `?start=checklist`
- Бот при старте с этим параметром сразу отправляет PDF

#### 7. Часто задаваемые вопросы (FAQ)
- «Нужен ли опыт?» → Нет, всё с нуля
- «Какие инструменты нужны?» → Только Grok и CapCut (оба бесплатные/дешёвые)
- «Как долго идёт курс?» → Можно пройти за 1–3 дня
- «Как получить доступ?» → Сразу после оплаты — в Telegram боте
- «Есть ли возврат?» → Да, в течение 3 дней если не прошёл ни одного урока

#### 8. Финальный CTA
- Кнопка «Начать обучение — 3 990 ₽» → бот

### Дизайн лендинга

Консистентен с существующим `player.html` (Tailwind + Inter). Не изобретаем новое.

**Цветовая схема:**
- Hero-секция: тёмный градиент `from-slate-900 to-violet-950` (premium AI-feel)
- Акцент: `violet-600` / `violet-700` — совпадает с цветами в `player.html`
- Остальные секции: белый фон, тёмно-серый текст (`gray-900`, `gray-600`)
- CTA-кнопка: `bg-violet-600 hover:bg-violet-700 text-white`

**Шрифт:** Inter (уже в player.html) — подключить через Google Fonts в `<head>`

**Стек:** Tailwind CDN (уже используется везде) — никаких новых зависимостей

**UX-принципы:**
- Mobile-first (аудитория из Instagram/TikTok)
- Один главный CTA (не несколько конкурирующих кнопок)
- Видео НЕ автозапускается — пользователь нажимает сам
- Чистые секции с вертикальным воздухом (`py-16` / `py-24`)

### Технические требования
- Шаблон: `app/templates/landing.html` (новый)
- Маршрут: `GET /` в `app/main.py`
- Статика: `app/static/landing/` — демо-видео, фото автора
- Ссылка на бот: `https://t.me/{settings.BOT_USERNAME}` (из config)
- Deep link для чек-листа: `https://t.me/{BOT_USERNAME}?start=checklist`
- Стек: Jinja2 + Tailwind CDN + Inter (консистентно с player.html)
- Адаптивность: мобильный в первую очередь (основная аудитория — Instagram/TikTok)

### Бот — обработка deep link
```python
# В обработчике /start
if message.text and "checklist" in message.text:
    # Сразу отправить PDF, затем follow-up
```

## Feature 7 — Трафик из Instagram и TikTok

> Это маркетинговая операция, не техническая задача.

### Текущая ситуация
Instagram — основной канал трафика, TikTok — вторичный. Это правильная стратегия: короткое демо-видео = лучший формат рекламы для AI-видео курса (контент сам себя продаёт).

### Контент-стратегия (работает прямо сейчас)

**Формат постов/Reels/Shorts:**
1. «Вот что я сделал в Grok за 5 минут» — показ процесса + результат
2. «3 типа AI-видео, которые делают миллионы просмотров» — польза + демо
3. «Как AI меняет создание контента в 2026» — образовательный
4. «До/После: обычное фото → AI-анимация» — трансформация

**CTA в каждом видео:**
- Instagram: «Ссылка в шапке профиля» (ведёт на лендинг или бот)
- TikTok: «Ссылка в описании» (ведёт на лендинг)

**Лендинг как промежуточная страница:** Instagram → Лендинг → Бот → Покупка
(лендинг объясняет курс тем, кто пришёл из соцсетей впервые)

### Периодичность
- Минимум 2–3 коротких видео в неделю (Reels + TikTok одновременно)
- Каждое видео = демонстрация того, что можно сделать после курса

### Метрики

| Метрика | Цель (месяц 1) |
|---------|---------------|
| Новых пользователей в боте | 100+ |
| Конверсия лендинг → бот | 10%+ |
| Конверсия бот → покупка | 3%+ |
| Продаж в месяц | 10+ |
